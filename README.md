スーパーマーケットライフのポイントデーを雑に表示するプログラム

### 使い方

* [店舗一覧](http://www.lifecorp.jp/store/)から対象店舗を調べる
* 対象店舗のキャンペーンの日付を`pointDays.json`に入力する
    * 曜日を指定する場合は`x曜日`の形式で`weekDays`に入力する
* `node index.js`を実行する
